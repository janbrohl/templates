Dies ist das SVN repository für Kalender templates.

Links ins repository bitte (außer zum Testen) nicht direkt mit dem Kalender verwenden, sondern die Datei angeben.

 RICHTIG: http://kalender.scoutnet.de/2.0/show.php?template=scoutnet.tpl

 FALSCH:  http://kalender.scoutnet.de/2.0/show.php?template=https://kalender.scoutnet.de/ ... /scoutnet.tpl

Wenn du weist, wie man mit SVN umgeht und dieses Repository nutzen möchtest, um deine templates direkt auf dem Kalender-Server zu verwalten sende bitte eine E-Mail an kalender@scoutnet.de .
